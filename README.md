# dom_manipulation

No framework. No React. Raw HTML, CSS, and JavaScript only.
Your tasks:
Task 1 — Render the transactions array from day1 into an HTML table using JavaScript. Not hardcoded HTML — your JS must build the table rows dynamically.
Task 2 — Add an input field where the user types a minimum amount and clicks a button. The table updates to show only transactions above that amount.
Task 3 — Add a line below the table that shows the total amount of the currently visible transactions. It must update every time the filter changes.
